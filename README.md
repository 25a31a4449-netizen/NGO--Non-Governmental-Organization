# 🌍 Green Earth NGO

A comprehensive PHP/MySQL platform for environmental organizations to manage volunteers, track donations, organize events, and engage communities.

## ✨ Features
- **User System**: Register/Login with secure authentication
- **Donations**: Online contributions with fund tracking
- **Volunteers**: Log hours and view leaderboards
- **Events**: Interactive listings and registration
- **Blog**: Dynamic posts with categories and search
- **Gallery**: Responsive image showcase
- **Contact**: Multi-purpose contact forms

## 🛠️ Tech Stack
PHP 7.4+, MySQL, HTML5, CSS3, JavaScript  
**Security**: Password hashing, prepared statements, CSRF protection
